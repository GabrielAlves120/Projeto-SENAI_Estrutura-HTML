Inserindo Links

Vamos criar um link para que o nosso usuário conheça o SENAI. O comando para criarmos um link é a Tag a. Se definirmos a target como:
_blank: abre a página em uma nova janela/aba;
_self: abre a página na mesma janela;
_parent: abre a página na mesma janela do link;

Tabelas

A tabela é uma estrutura composta de linhas, com inúmeras possibilidades de formatação. Para construir uma tabela usamos a Tag table, para construir uma linha usamos a Tag tr, e para criar uma coluna usamos a Tag td

Imagem

É comum querermos ou precisarmos inserir imagens na nossa página web. Isto deixa o seu site mais atraente e se você está desenvolvendo uma loja virtual isso é essencial. Para isso utilizamos a Tag img. Podemos alterar sua altura e largura usando os atributos width(largura) e height(altura)
